RODE Wireless PRO + GO II
4人ピンマイク収録 構成図

前提：1台の受信機に接続できる送信機は2台まで。
4人を同時収録するには、受信機が2台必要です。
ここでは、Macで4トラック別録りする本命構成と、iPhoneで手軽に録る簡易構成を並べて示します。

おすすめ本命構成：Mac収録

Macで4人を別トラック収録

編集の自由度が最も高く、後処理が圧倒的に楽な構成です。

話者Aラベリア → Wireless PRO TX1
話者Bラベリア → Wireless PRO TX2
→
Wireless PRO RX2人分を受信
設定：Split
→
MacUSB-Cで接続
入力1–2として認識
話者Cラベリア → Wireless GO II TX1
話者Dラベリア → Wireless GO II TX2
→
Wireless GO II RX2人分を受信
設定：Split
→
MacUSB-Cで接続
入力3–4として認識
Mac側の考え方
Audio MIDI設定で Aggregate Device を作成し、2台の受信機を1つの入力機器として束ねます。 録音ソフトでは各話者をモノラルトラック4本に割り当てます。

メリット

  • 4人を個別トラックで録れる
  • ノイズ処理・音量調整・カット編集がしやすい
  • 一人だけ音が小さい時も救いやすい

必要なもの

  • Mac
  • 受信機2台をつなぐUSBケーブル
  • 4人分のラベリア
  • 録音アプリ

事故防止の保険

  • 4台すべての送信機でオンボード録音をON
  • PROは重要話者に回す
  • 本番前に各人の入力レベルと録音動作を確認
手軽な簡易構成：iPhone収録

iPhoneで4人をまとめて収録

接続はシンプルですが、ライブ時点では4人完全別トラックにはなりにくい構成です。

話者Aラベリア → Wireless PRO TX1
話者Bラベリア → Wireless PRO TX2
→
Wireless PRO RX設定：Merge
↘
iPhoneUSB-Cまたはアダプタ経由で入力
実運用は2系統ミックス
話者Cラベリア → Wireless GO II TX1
話者Dラベリア → Wireless GO II TX2
→
Wireless GO II RX設定：Merge
↗
2受信機を1入力へ
SC11や対応IFで合流
iPhone側の考え方
2台の受信機をまとめて1つの入力へ入れます。
そのため、収録時の見え方は「4人別々」ではなく、実質は次のような2系統になりやすいです。
左または系統1
話者A + 話者B
右または系統2
話者C + 話者D

メリット

  • 機動力が高い
  • 現場セットアップが速い
  • カメラ代わりにそのまま動画収録しやすい

注意点

  • 4人完全別トラック運用には向かない
  • 一人だけの修正がしにくい
  • 最終品質は内部録音の回収が前提になりやすい

事故防止の保険

  • 4台すべての送信機でオンボード録音をON
  • あとで必要なら各TXの内部録音をMacへ吸い上げて救済
  • その場で簡単に録りたい時の方法と割り切る
最終判断

どちらを選ぶべきか

Macを選ぶべきケース

  • ポッドキャスト
  • 対談・座談会
  • あとでちゃんと編集したい
  • 4人を別々に整音したい

iPhoneを選ぶべきケース

  • 現場で素早く撮りたい
  • 動画も同時に押さえたい
  • 編集を最小限にしたい
  • 内部録音を保険として使える

ひとことで言うと、完成品質優先ならMac、手軽さ優先ならiPhoneです。
ただし、4人を後でしっかり扱う前提なら、結局は Mac + 4トラック別録り が最も効率的です。
