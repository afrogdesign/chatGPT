Macオーディオの要点整理

Aggregate Device とは何か

Aggregate Device は、Macで複数のオーディオ機器を 1つの仮想オーディオ機器として束ねる機能です。
今回のように Wireless PRO RX と Wireless GO II RX を同時に使う時、 Mac側からはそれらを 4ch入力の1台 のように扱えるようになります。

まず結論

何のために使うのか

  • USBオーディオ機器が2台ある
  • でも録音アプリは通常、入力機器を1つ選ぶ前提
  • そこで Mac が2台をまとめて1台の入力機器に見せる

今回の意味

  • PRO受信機 = 2入力
  • GO II受信機 = 2入力
  • Aggregate Device = 合計4入力として扱える

なぜ便利なのか

1. 4人を別トラックで録れる

あとで一人ずつ音量調整・EQ・ノイズ除去ができます。

2. 事故に強い

一人だけ小さい、咳が入った、ノイズが出た、という時に救いやすくなります。

3. 編集が圧倒的に楽

全員まとめ録りよりも、カット編集・整音・被り処理がしやすいです。

似ている機能との違い

Aggregate Device

  • 複数の入力機器を1つに束ねる
  • 入力チャンネル数を増やせる
  • 録音・配信・DAW向き

Multi-Output Device

  • 複数の出力先へ同時に再生する
  • スピーカーやヘッドホンの同時出力向き
  • 録音入力をまとめる用途ではない

作り方

  1. Wireless PRO RX と Wireless GO II RX を、USBでMacへ接続する
  2. Macの Audio MIDI設定 を開く
  3. 左下の ＋ を押して Aggregate Deviceを作成 を選ぶ
  4. 一覧で PRO RX と GO II RX にチェックを入れる
  5. どちらか1台を Clock Source にする
  6. もう片方の機器には Drift Correction を入れる
  7. サンプルレートをできれば 48kHz にそろえる
  8. 録音アプリで Aggregate Device を入力機器に選び、4本のモノラルトラックへ割り当てる

Clock Source と Drift Correction の意味

Clock Source

2台のオーディオ機器は、内部の時計がわずかにズレます。
そこで1台を基準時計にして、全体の同期の基準にします。

Drift Correction

基準ではない側のわずかなズレを、Mac側で補正する機能です。
長時間録音でズレたり、プチプチノイズが出たりするのを防ぎやすくなります。

注意点

チャンネル順は必ず確認

Input 1〜4 が誰に対応しているか、本番前に声出しや拍手で確認した方が安全です。

サンプルレートはそろえる

44.1kHz と 48kHz が混ざると不安定になりやすいです。

録音アプリによって扱いやすさが違う

4トラック別録りなら、Logic Pro、Reaper、Audition、OBS系の方が扱いやすいです。

おすすめ実運用

RODE側

  • PRO RX = Split
  • GO II RX = Split
  • 全送信機 = オンボード録音ON

Mac側

  • Aggregate Device 作成
  • Clock Source を1台決める
  • もう1台に Drift Correction
  • 48kHz で統一

録音ソフト側

  • 4本のモノラルトラックを作る
  • Input 1〜4 をそれぞれ割り当てる
  • 録音前に全員分テストする

一言でまとめると

Aggregate Device は、2台のRODE受信機を、Mac上で1台の4ch入力機器として扱うための仕組みです。
4人を後でしっかり編集したいなら、Mac + Aggregate Device + 4モノラルトラック が最も効率的です。

補助的な考え方としては、Mac収録が本番、各送信機の内部録音が保険、という二重化が最も事故に強い構成です。
