## 🎯 まず結論

| 項目 | 採用案 |
|---|---|
| LP公開サーバー | **Xserver スタンダード** |
| APIサーバー | **Cloudflare Workers Paid** |
| 音源保存 | **Cloudflare R2** |
| 軽量ログ/一時情報 | **Workers KV の無料枠内運用** |
| 音楽生成API | **Mubert API** |
| 日本語入力 | 対応する |
| API送信 | 内部で英語プロンプトへ変換 |
| 1000人規模の推奨 | **これで十分** |

### この構成の意味

- **Xserver** は LP を安定して見せる担当
- **Workers** は 日本語入力を英語へ整えて Mubert に送る担当
- **R2** は 生成された音源を一時保存する担当
- **Mubert** は 実際に音を作る担当

> [!tip]
> すべてを1台のサーバーで無理にやるより、  
> **表示・処理・保存・生成を分けたほうが安くて壊れにくい**です。

---

## 🤖 Mubert API を使う理由

Mubert API は、短い音楽生成を Web サービスやコンテンツに組み込む用途と相性が良いです。

### 向いている理由

| 理由 | 説明 |
|---|---|
| 短尺生成に強い | 15秒以上の短い音に向いている |
| API前提で扱いやすい | LPやキャンペーンに組み込みやすい |
| 費用が読みやすい | generation数で管理しやすい |
| インスト用途に合う | 今回の「アートっぽい短音源」に向いている |
| 歌ものより安全 | 権利や声の問題を少し避けやすい |

## 🈯 日本語対応の考え方

### 結論

**ユーザーは日本語で入力してよい**ですが、内部では英語に変換して Mubert に送るのが安全です。

```text
日本語入力
↓
意味を整理
↓
英語プロンプト化
↓
Mubert APIへ送信
```

### 例

#### ユーザー入力

```text
夜、雨、静けさ、余白
```

#### API送信用の内部英文

```text
Create a 15-second ambient instrumental track with a quiet rainy night atmosphere, spacious texture, soft emotional tone, no vocals.
```

### この方式の利点

| 項目 | 理由 |
|---|---|
| 日本語UI | ユーザーが自然に使える |
| 品質安定 | 英語で送るほうが結果が安定しやすい |
| 修正しやすい | 変換辞書を調整するだけで改善できる |
| 安全性 | 禁止語や危険ワードを途中で弾ける |

## 🏗️ 全体構成

```text
[Xserver上のLP]
  ↓
[フォームから日本語入力]
  ↓
[Cloudflare Workers]
  ├─ 入力チェック
  ├─ 日本語→英語変換
  ├─ Mubert API 呼び出し
  ↓
[Cloudflare R2]
  ├─ 音源の一時保存
  ↓
[LPで再生]
```

### この構成が良い理由

| 担当 | サービス | 理由 |
|---|---|---|
| LP表示 | Xserver | 普通のWeb公開に強い |
| API処理 | Workers | 少人数〜中規模に安く強い |
| 音源保存 | R2 | egress無料で相性が良い |
| 音楽生成 | Mubert | 今回の用途に合う |

## 🖥️ LPを置く Xserver の選定

### 選定結果

**Xserver スタンダード** を推奨します。

### 理由

- LP本体を置くだけなら十分
- 1000人規模では過剰投資になりにくい
- SSLや通常のサイト公開機能が揃っている
- コストがかなり安い

### 今回の判断

今回のLPは、重い動画配信サービスや会員制サイトではなく、**キャンペーンLP + 外部API連携**です。  
そのため、Xserver側に高負荷処理を載せない構成にすれば、スタンダードで問題ありません。

### Xserver の現行料金メモ

| 契約期間 | スタンダード月額 |
|---|---:|
| 3ヶ月 | 1,320円 |
| 6ヶ月 | 1,210円 |
| 12ヶ月 | 1,100円 |
| 24ヶ月 | 1,045円 |
| 36ヶ月 | 990円 |

### 推奨契約

| 方針 | おすすめ |
|---|---|
| 短期案件として分かりやすく始める | **12ヶ月 1,100円/月** |
| とにかく固定費を下げる | 36ヶ月 990円/月 |

> [!info]

## ☁️ APIサーバーと保存先の選定

## 1. APIサーバー

### 選定結果

**Cloudflare Workers Paid** を推奨します。

### 理由

- 最低 **$5/月** から始められる
- 10 million requests / month が含まれる
- 小規模〜中規模のフォーム処理にかなり強い
- 運用が軽い
- Xserver と役割を分けやすい

### 今回の用途に合う処理

- 入力チェック
- 禁止語チェック
- 日本語→英語変換の呼び出し
- Mubert API 呼び出し
- 生成結果の返却

## 2. 音源保存先

### 選定結果

**Cloudflare R2** を推奨します。

### 理由

- 10GBまで無料枠がある
- 1 million Class A / 10 million Class B の無料枠がある
- egress が無料
- Workers と相性がよい

### 今回の用途に合う理由

- 生成した mp3 を一時保存できる
- 期間限定の配信に向く
- 小規模ならほぼ無料で済みやすい

## 3. DBはどうするか

### 初期結論

**1000人規模なら、最初は重いDBを持たなくてよい**です。

- 一時情報は Workers KV
- 音源本体は R2
- 本格的な管理画面が必要になったら D1 / Supabase を追加

> [!tip]
> 最初からDBを重くすると、  
> 1000人規模ではコストより構成の複雑さのほうが先に増えます。

## 💰 サーバー費の見積り

## 前提

- 利用者: 1000人
- LPは Xserver 上で公開
- APIは Cloudflare Workers
- 音源保存は R2
- 同時接続1000人ではなく、全体利用1000人想定

## 月額の目安

| 項目 | 選定 | 月額目安 |
|---|---|---:|
| LPサーバー | Xserver スタンダード 12ヶ月換算 | 1,100円 |
| APIサーバー | Cloudflare Workers Paid | 約800円 |
| 音源保存 | Cloudflare R2 | 0円〜数百円程度 |
| 一時情報 | Workers KV | ほぼ0円想定 |

### サーバー周りの結論

**サーバー関連だけなら、月額おおむね 1,900円前後で開始可能**です。

### より安全な見積り

R2 の利用量や為替のブレを見込むと、

| 見積タイプ | 月額想定 |
|---|---:|
| 最小 | 約1,900円 |
| 安全見積 | **2,500〜3,000円程度** |

> [!note]
> 1000人規模では、サーバー代よりも **Mubert API代** のほうが支配的になります。

## ⚠️ 注意点

### 1. Xserver に重い処理を載せない

LPだけを置き、API処理は外に逃がすほうが安定します。

### 2. 日本語はそのまま Mubert に投げない

内部で英語化するほうが安全です。

### 3. 有名アーティスト名は禁止する

### 4. ダウンロードは弱くする

「その場で聴く体験」に寄せたほうが事故が少ないです。

### 5. 1000人同時接続なら別設計

今回は全体利用1000人前提です。  
もし**同時に1000人が一斉アクセス**するなら、見積と構成は別になります。
